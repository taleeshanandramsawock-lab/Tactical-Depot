## Project Description

Tactical Depot is an e-commerce website designed for selling tactical gear and firearms. The website provides an online platform for customers to browse products, view contact information, and submit orders through a user-friendly interface.

---

## Technologies Used

- **HTML5** - Page structure and content
- **CSS3** - Styling and layout
- **Bootstrap 5** - Responsive design framework
- **MySQL** - Database management system

## Website Features

- Responsive navigation menu

- Contact information page
- Customer order form with dropdown selections
- Mobile-friendly design

---

## Database Tables

1. **Customers** - Stores customer information
2. **Products** - Contains tactical gear product catalog
3. **Orders** - Records customer orders
4. **OrderDetails** - Stores individual order items
5. **Employees** - Employee and admin login data

---

## How to Run the Project

### Frontend (HTML/CSS):
1. Download or clone this repository
2. Open `html/index.html` in a web browser
3. Navigate using the menu

### Backend (Database):
1. Install MySQL or XAMPP
2. Open phpMyAdmin or MySQL Workbench
3. Import the file `sql/database.sql`
4. Database and tables will be created automatically
